# NoopJennExternalPlugin

- This is an inert no-op external plugin placeholder.
- It exists only to prove external package plugin shape.
- It is not migrated code.
- It must not override any core plugin.
- It must not use secrets, LocalState, logs, cache, outputs, or providers.
- It must not be installed through Plugin Store in Gate 14.
- It must not be treated as production plugin.
- Future activation requires a separate gate.
