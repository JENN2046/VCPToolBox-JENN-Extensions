# DWS tool scripts

- `run-real-baseline.js`: run version/auth/schema baseline checks and write reports.
- `calibrate-schema-map.js`: discover command tree from `dws --help` and generate capability map.
- `run-capability-matrix.js`: execute 10-capability query/write matrix through DingTalkCLI.
- `run-workflow-e2e.js`: execute built-in workflows with apply=false/true.
